# HR الابداع

## Requirements

Before you begin, ensure you have the following requirements installed on your machine:

### Software Requirements

- [PHP](https://www.php.net/downloads/) (version 7.x or higher)
- [Composer](https://getcomposer.org/download/) (for managing libraries)
- A local server (e.g., [XAMPP](https://www.apachefriends.org/index.html), [WAMP](http://www.wampserver.com/en/), or [MAMP](https://www.mamp.info/en/)) to run the PHP application.

## Installing Libraries

You can update and install the necessary libraries using Composer. Open your command line in the project directory and run:

```bash
composer update
